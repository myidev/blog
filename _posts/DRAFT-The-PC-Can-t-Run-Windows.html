 Mengatasi The PC Can’t Run Windows 11 – Ada dua cara untuk memperbaiki kesalahan The PC Can’t Run Windows 11” dan kesalahan terkait driver yang mungkin muncul selama proses instalasi. Pertama, untuk mem-boot Windows 11, Microsoft telah mewajibkan dukungan TPM 2.0 dan Secure Boot. Jadi pada dasarnya, Anda harus boot ke BIOS/UEFI dan mengaktifkan TPM dan Boot Aman. Dan kemudian, penginstal Windows 11 akan dimuat tanpa masalah.

Namun, saya tidak mengambil rute ini karena Secure Boot mungkin dapat menghentikan instalasi Windows 11 karena pemeriksaan integritas dan tanda tangan. Jadi kami memiliki solusi lain, yang merupakan metode yang dicoba dan diuji, dan berhasil berhasil di komputer saya.

1. Klik tautan ini dan unduh Image ISO Windows 10 terbaru dari situs web Microsoft. Saya menggunakan build Windows 10 Mei 2021, jadi saya akan merekomendasikannya.

2. Setelah Anda mengunduh gambar ISO Windows 10, klik kanan padanya dan  select “Mount”.


3. Gambar ISO akan dipasang sebagai drive. Sekarang, pindah ke drive yang terpasang dan buka folder “Source”.

4. Di sini, pilih semua file kecuali untuk “install.wim” Atau “install.esd“ Sekarang, tekan “Ctrl + C” untuk menyalin file. Ulangi lagi, pastikan “install.wim” atau “install.esd” tidak dipilih saat Anda menyalin file.

5. Sekarang, colokkan USB thumb drive tempat Anda menginstal Windows 11. Buka drive USB dan cari ke folder “Source”

6. Kemudian pastekan Semua File yang tadi anda Copy di folder tersebut.

7. Setelah file diganti, Anda hampir selesai. Sekarang, restart saja komputer Anda dan ikuti instruksi yang sama yang telah saya sebutkan di bagian di atas. Kali ini Anda tidak akan menghadapi “The PC Can’t Run Windows 11” ? atau kesalahan terkait Driver saat menginstal Windows 11 di PC Anda.
